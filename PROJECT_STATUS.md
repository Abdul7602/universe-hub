# Universe Hub

## Vision

Universe Hub is a cinematic portfolio experience built as an interactive digital universe.

Visitors explore professional identity, projects, GitHub activity, career history, and contact information through celestial destinations rather than traditional pages.

---

## Tech Stack

- Next.js 16
- TypeScript
- Tailwind CSS
- Three.js
- React Three Fiber
- Framer Motion
- GSAP
- Zustand (planned)
